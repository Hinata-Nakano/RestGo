# How to setup
  ## githooksの設定
    1. Setting up Git Hooks
   
  - run the below command in shell. This command makes git use .githooks as git hook automatically.
   ```shell
      $ chmod +x setup.sh 
      $ ./setup.sh
  ```

  ## 開発環境の統一
    1. vscodeでDev Containersをダウンロードする
    2. Reopen in containerを選択し、RestCRUDフォルダを開く
